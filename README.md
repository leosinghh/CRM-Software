# CRM-Software
A CRM-style dashboard where brands and influencers manage deals, contracts, deliverables, and real-time performance in one place.
Key features

Automatic contract builder for brand deals (usage rights, deliverables, timelines).

Negotiation messenger with templates for influencers to ask for better rates.

AI pricing benchmark analyzing niche, audience country, prior performance.

Campaign Tracker showing exact ROI for each collaboration.

Shared Workspace where both sides see deadlines, drafts, approvals.

Auto-invoice + payout management
